Place your textbook PDF files in this 'data' folder.

Supported Formats:
- PDF (.pdf) - Scans page by page, preserving original page references.
- Text (.txt) - Processes raw text paragraphs.
- Markdown (.md) - Processes raw markdown content.

Steps to Run:
1. Make sure your PostgreSQL database is running and the DATABASE_URL is set in your .env.
2. Put a textbook PDF file in this directory.
3. Run: python python_backend/offline_setup.py (or change directory to python_backend/ and run: python offline_setup.py)
4. The script will parse the PDF, generate embeddings, and populate PostgreSQL.
